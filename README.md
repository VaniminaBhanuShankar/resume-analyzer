# Resume Analyzer

## Description

This project is a Resume Analyzer built using Python.
It analyzes resumes and extracts useful information like skills and keywords.


## Features

* Upload and analyze resumes
* Extract important keywords
* Simple interface


## Technologies Used

* Python
* Flask
* HTML / CSS
